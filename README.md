#Luna Platform 

A compassionate platform designed to help university students explore their cognitive patterns (ADHD, Autism, etc.) through gentle interaction and AI-driven insights.

## 🚀 Key Features
- **Empathetic Quiz:** Questionnaires designed with friendly emojis to reduce testing anxiety.
- **Behavioral Task:** A "Time Magic Challenge" to detect time perception differences (common in ADHD/Autism).
- **AI Analysis:** Uses Google Gemini AI to provide supportive, personalized feedback.
- **Safe Space:** Integrated emergency contact triggers for immediate support.

## 🛠️ Tech Stack
- **Frontend:** Streamlit (Python)
- **AI Engine:** Google Generative AI (Gemini)
- **Security:** Python-dotenv for API protection.

## 🔒 Setup Instruction
1. Clone the repository.
2. Install dependencies: `pip install -r requirements.txt`
3. Create a `.env` file and add your `GOOGLE_API_KEY`.
4. Run the app: `streamlit run app.py`
